# Property Document Upload Samples

Sample PDFs are grouped to match the six document upload sections in the developer property submission flow.

## Folder Mapping

| Upload section | Folder | Sample documents |
| --- | --- | --- |
| 1. Proof of title | `01-proof-of-title/` | HGB certificate, Hak Pakai certificate, HMSRS certificate |
| 2. Legal basis of title | `02-legal-basis-of-title/` | Deed of Sale and Purchase (AJB), Leasehold Agreement |
| 3. Permits (existing structures) | `03-permits-existing-structures/` | Building Approval (PBG / IMB) |
| 4. Tax documentation | `04-tax-documentation/` | SPPT PBB payment proof, BPHTB |
| 5. KYC / Corporate structure | `05-kyc-corporate-structure/` | Owner passport, Deed of Establishment, SK Kemenkumham approval, NIB/OSS, NPWP |
| 6. Declarations | `06-declarations/` | No Dispute Declaration |

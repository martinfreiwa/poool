# Developer Expenses

This folder tracks recurring developer and operating expenses for running the POOOL website.

## Files

| File | Description |
| --- | --- |
| `2026-05.md` | May 2026 expense summary |
| `expenses.csv` | Machine-readable expense ledger |

## Rules

- Store money as integer cents in structured files.
- Keep monthly summaries in USD unless an invoice is charged in another currency.
- Add the invoice month, vendor, category, description, and amount for every recurring cost.
